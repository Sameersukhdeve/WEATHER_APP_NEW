=== WHAT YOU NEED TO DO ===

1. ⏳ Get an OpenWeatherMap API Key
   - Go to: https://openweathermap.org/api
   - Sign up (free tier available)
   - Copy your API key

2. ⏳ Add API Key to .env.local
   - Edit: .env.local
   - Replace: OPENWEATHER_API_KEY=your_openweathermap_api_key_here
   - With your actual API key

3. ⏳ Test Locally (Optional)
   - Terminal 1: cd backend && npm start
   - Terminal 2: npm start
   - Open: http://localhost:3000
   - Test weather search

4. ⏳ Push to GitHub
   - git add .
   - git commit -m "Setup SQLite and fix Vercel deployment"
   - git push origin main

5. ⏳ Deploy to Vercel
   - Go to: https://vercel.com/new
   - Import your GitHub repo
   - Select "weather-app" as root directory
   - Add environment variable: OPENWEATHER_API_KEY=your_key
   - Click Deploy

6. ✨ Done!
   - Your app will be live at: https://weather-app-virid-omega-53.vercel.app/
   - (or your Vercel domain)

=== SUPPORT ===

If you get errors:

404 Error:
→ Make sure vercel.json has the "routes" section

OpenWeatherMap API Key Error:
→ Check .env.local has OPENWEATHER_API_KEY set

Database Error:
→ SQLite file will auto-create at ./data/weather.db

Backend won't start:
→ Make sure port 5000 is free
→ Check Node.js is installed: node --version
